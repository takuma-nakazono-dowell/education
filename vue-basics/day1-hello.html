Vue.js とは？

Vue.js（ビュージェイエス）は、WebページのUIをかんたんに構築できる JavaScriptフレームワークです。 「リアクティブ」なUI、つまりデータが変わると画面が自動的に更新される 仕組みを提供します。

Vue.js の特徴
・学習コストが低く、HTMLに近い書き方ができる
・データと画面を自動的に同期させる（リアクティブ）
・コンポーネント単位でUIを組み立てられる
・CDNで読み込むだけで使い始められる

Vanilla JS との比較

同じ「ボタンを押すたびにカウントを増やす」処理を比較してみましょう。

Vanilla JS（従来の書き方） Vue.js の書き方
DOMを直接操作する
document.getElementById(...)		 データを操作するだけ
this.count++
画面の更新コードも書く必要がある Vueが自動で画面を更新してくれる
コードが長くなりがち コードが短くシンプルになる
// ❌ Vanilla JS: DOMを直接操作する let count = 0; const btn = document.getElementById('btn'); const display = document.getElementById('display'); btn.addEventListener('click', () => { count++; display.textContent = count; // 手動で更新が必要 }); // ✅ Vue.js: データだけ変えればOK Vue.createApp({ data() { return { count: 0 } }, methods: { increment() { this.count++ } // 画面は自動更新！ } }).mount('#app')

Vue 3 のCDN導入

Vue.jsはCDNから読み込むだけで使えます。 HTMLファイルに以下の1行を追加するだけです。

<!-- </body> の直前に追記する --> <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

createApp() と mount()

Vueアプリの作り方は2ステップです。

// Step 1: Vueアプリを作る const app = Vue.createApp({ // ここにオプションを書く data() { return { message: 'こんにちは、Vue.js！' } } }) // Step 2: HTMLの要素にマウント（取り付ける） app.mount('#app')
mount() とは？
mount('#app') は「id="app" の要素にVueアプリを適用する」という意味です。 セレクタ文字列（CSSセレクタ形式）を渡します。

テンプレート構文 {{ }}

HTMLの中で {{ データ名 }} と書くと、 Vueのデータをそこに表示できます。これをマスタッシュ構文（mustache）と呼びます。

<div id="app"> <h1>{{ message }}</h1> <p>1 + 1 = {{ 1 + 1 }}</p> <p>文字数: {{ message.length }}</p> </div>
{{ }} の中に書けるもの
・変数名: {{ message }}
・式（計算）: {{ price * 1.1 }}
・メソッド呼び出し: {{ message.toUpperCase() }}
・三項演算子: {{ isOk ? 'はい' : 'いいえ' }}

data() 関数でデータを定義する

Vue.createApp({ data() { return { // ここに表示したいデータを書く name: '田中太郎', age: 25, isActive: true } } }).mount('#app')
注意: data() は関数で書く
data: { ... }（オブジェクト）ではなく、 data() { return { ... } }（関数）で書きます。 Vue 3ではこれが正しい書き方です。

動くデモ & 練習問題

デモ: Vue.js の基本動作を確認しよう

下のデモはVue.jsで動いています。データが画面に表示される様子を確認してください。

{{ message }}

カウント: {{ count }}

名前: {{ name }}（{{ age }}歳）

問題 1: テンプレート構文を読んでみよう

以下のVueアプリのコードがあります。
data() { return { message: 'Hello Vue!', count: 10 } }
HTMLに <p>{{ message }} / {{ count }}</p> と書いたとき、 何が表示されますか？
答えを見る
Hello Vue! / 10 と表示されます。
{{ message }} は "Hello Vue!" に、 {{ count }} は 10 に置き換わります。

問題 2: 式の評価

以下のテンプレートはそれぞれ何を表示しますか？
// data: { price: 1000, tax: 0.1 } <p>{{ 1 + 1 }}</p> <p>{{ price * (1 + tax) }}</p> <p>{{ 'Vue' + '.js' }}</p>
答えを見る

{{ 1 + 1 }} → 2

{{ price * (1 + tax) }} → 1100（1000 × 1.1）

{{ 'Vue' + '.js' }} → Vue.js（文字列の結合）

問題 3: バグを見つけよう

以下のコードはうまく動きません。どこが間違っているか分かりますか？
<div id="myApp"> <p>{{ message }}</p> </div> Vue.createApp({ data() { return { message: 'こんにちは' } } }).mount('#app') // ← ここを確認！
答えを見る
mount('#app') が間違いです。
HTMLの要素は id="myApp" なので、 mount('#myApp') と書く必要があります。

mountのセレクタはHTMLのidやclassと一致させてください。

問題 4: 自分で書いてみよう

「名前: 山田花子、年齢: 22歳、趣味: 読書」を表示するVueアプリを 完成させてください。___ の部分を埋めてみましょう。
<div id="app"> <p>名前: {{ ___ }}</p> <p>年齢: {{ ___ }}歳</p> <p>趣味: {{ ___ }}</p> </div> Vue.createApp({ data() { return { ___: '山田花子', ___: 22, ___: '読書' } } }).mount('#app')
答えを見る
<div id="app"> <p>名前: {{ name }}</p> <p>年齢: {{ age }}歳</p> <p>趣味: {{ hobby }}</p> </div> Vue.createApp({ data() { return { name: '山田花子', age: 22, hobby: '読書' } } }).mount('#app')
変数名は自由につけられますが、HTMLのテンプレートとdata()で 同じ名前を使うことがポイントです。
