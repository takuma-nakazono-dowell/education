今日作るもの

Day 1〜6 で学んだすべての機能を使って、本格的なTODOアプリを作ります。 右パネルで実際に動くデモを確認しながら、コードを読み解いていきましょう。

使う機能一覧
・v-model — テキスト入力
・v-for — タスクリストの繰り返し表示
・v-if / v-show — 条件による表示切り替え
・@click / @keyup.enter — イベント処理
・computed — 残りタスク数・フィルタリング
・watch — ローカルストレージ自動保存

実装のステップ

Step 1: データ設計
どんなデータが必要か考える。タスクの配列、入力中のテキスト、フィルター状態など。
Step 2: タスクの追加
v-model で入力を受け取り、ボタン or Enter で配列に追加する。
Step 3: タスクの表示
v-for でタスク配列をループして一覧表示する。
Step 4: 完了チェック
チェックボックスと v-model で done フラグを切り替える。
Step 5: 削除 & 一括削除
methods で配列から対象を filter で除去する。
Step 6: computedでフィルタリング
「全件 / 未完了 / 完了済み」の3種類を computed で絞り込む。
Step 7: watchで自動保存
todos 配列を watch して変化のたびに localStorage に保存する。

完成コードの全体像

// ===== data ===== data() { return { newTask: '', // 入力中のタスク名 filter: 'all', // 'all' | 'active' | 'done' nextId: 1, todos: [] // タスクの配列 } }, // ===== computed ===== computed: { // フィルタ済みのタスク一覧 filteredTodos() { if (this.filter === 'active') { return this.todos.filter(t => !t.done) } else if (this.filter === 'done') { return this.todos.filter(t => t.done) } return this.todos }, // 残りタスク数（未完了のみカウント） activeCount() { return this.todos.filter(t => !t.done).length }, // 完了済みタスク数 doneCount() { return this.todos.filter(t => t.done).length } }, // ===== methods ===== methods: { addTask() { if (!this.newTask.trim()) return this.todos.push({ id: this.nextId++, text: this.newTask.trim(), done: false }) this.newTask = '' }, removeTask(id) { this.todos = this.todos.filter(t => t.id !== id) }, clearDone() { this.todos = this.todos.filter(t => !t.done) } }, // ===== watch ===== watch: { todos: { handler(newVal) { localStorage.setItem('todos', JSON.stringify(newVal)) }, deep: true // オブジェクトの深い変化も監視 } }

チャレンジ課題

TODOアプリを拡張してみましょう。難易度順に挑戦してください。

チャレンジ課題一覧
  • 1
    タスクに優先度を追加する
    タスクに「高 / 中 / 低」の優先度フィールドを追加し、 追加フォームにセレクトボックスを設ける。 表示時に優先度を色分けする（高=赤、中=橙、低=青など）。
  • 2
    優先度でフィルタリングできるようにする
    computed を使って、優先度で絞り込める機能を追加する。 フィルターボタンに優先度ボタンを追加し、 filteredTodos に優先度フィルタの条件を組み込む。
  • 3
    タスクの編集機能を追加する
    各タスクに「編集」ボタンを追加。 クリックするとタスクのテキストがインライン編集できる input に変わり、 Enterまたはフォーカス離脱で確定する。 v-if で表示と編集を切り替える。
  • 4
    ドラッグ & ドロップで並び替え（上級）
    HTML5の Drag and Drop API を使って、 タスクの順番を入れ替えられるようにする。 vue の @dragstart / @dragover / @drop イベントを活用する。
