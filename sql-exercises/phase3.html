応用のヒント

NULL の除外
WHERE height IS NOT NULL AND weight IS NOT NULL
height/weight は TEXT 型なので
CAST(height AS REAL) で数値変換
ROUND（四捨五入）
ROUND(値, 1) → 小数第2位四捨五入
日跨ぎの判定
date(sche_ope_start_time) != date(sche_ope_end_time)
件数最多を1件取り出す
GROUP BY → ORDER BY COUNT(*) DESC → LIMIT 1
存在しないレコードの抽出
LEFT JOIN + WHERE FK IS NULL
または NOT EXISTS (サブクエリ)
比率の計算
整数同士の割り算は 1.0 を掛けて浮動小数点にする
COUNT(*) * 1.0 / SUM(COUNT(*)) OVER(...)

Phase 3 — 応用・分析

問1: 手術が行われていない診療科

operation_table に手術実績が1件もない診療科を抽出してください。
出力列: section_name
ソート: idx_dsp 昇順

問2: BMI計算（身長・体重不明を除外）

各患者のBMIを計算してください。
身長または体重が不明（NULL）の患者は除くこと。
BMI = 体重(kg) ÷ 身長(m)²　小数第2位で四捨五入（小数第1位まで表示）
出力列: patient_name, height, weight, bmi
ソート: ope_date 昇順

問3: 午前中に件数が最多の診療科

午前中（AM11:59まで）に行われた手術で、一番件数が多い診療科はどこか？
出力列: section_name, 件数
※同率1位の場合はすべて表示

問4: 深夜緊急手術で件数が最多の診療科

深夜（PM10時以降〜AM5時）の緊急手術で、一番件数が多い診療科はどこか？
出力列: section_name, 件数
※同率1位の場合はすべて表示

問5: 日跨ぎの手術件数

開始日と終了日が異なる（日跨ぎ）手術は何件あるか？
出力列: 件数

問6: 利用頻度が最も高い手術室

一番利用件数が多い手術室はどこか？
出力列: ope_room_name, 件数
※同率1位の場合はすべて表示

問7: 女性比率が高い診療科

診療科ごとに女性患者の比率を計算し、比率が高い順に表示してください。
小数第2位で四捨五入（小数第1位まで表示）
出力列: section_name, 総件数, 女性件数, 女性比率(%)
ソート: 女性比率 降順
