DOMとは

DOM（Document Object Model）は、HTMLをJavaScriptから操作するための仕組みです。 ブラウザはHTMLを読み込むとツリー構造（DOMツリー）を作り、JavaScriptからアクセスできるようにします。

要素の取得

// id属性で取得（1つ） const el = document.getElementById("myId"); // CSSセレクタで取得（最初の1つ） const el = document.querySelector(".myClass"); const el = document.querySelector("#myId"); const el = document.querySelector("ul li:first-child"); // CSSセレクタで取得（すべて） const els = document.querySelectorAll("li"); els.forEach(el => console.log(el.textContent));

テキストとHTMLの操作

const el = document.getElementById("title"); // textContent: テキストのみ取得・設定（安全） console.log(el.textContent); el.textContent = "新しいタイトル"; // innerHTML: HTMLタグも含めて取得・設定 el.innerHTML = "<strong>太字</strong>のテキスト";
textContent と innerHTML の違い
・textContent: HTMLタグを文字として扱う（XSS対策に安全）
・innerHTML: HTMLとして解釈する（タグが実際に適用される）

スタイルの変更

const el = document.querySelector(".box"); // styleプロパティで直接変更（camelCase） el.style.color = "red"; el.style.backgroundColor = "#e3f2fd"; el.style.fontSize = "20px"; // クラスで管理（推奨） el.classList.add("active"); el.classList.remove("hidden"); el.classList.toggle("selected");

イベントリスナー

const btn = document.querySelector("button"); // クリックイベント btn.addEventListener("click", function() { console.log("ボタンがクリックされました"); }); // アロー関数でも書ける btn.addEventListener("click", () => { console.log("クリック！"); }); // inputイベント（入力のたびに発火） const input = document.querySelector("input"); input.addEventListener("input", (e) => { console.log(e.target.value); });

要素の動的追加

// 新しい要素を作る const li = document.createElement("li"); li.textContent = "新しい項目"; // 親要素に追加 const ul = document.querySelector("ul"); ul.appendChild(li);
注意
DOMを操作するJavaScriptは、HTMLが読み込まれた後に実行する必要があります。 scriptタグはbodyの末尾か、DOMContentLoadedイベント内で実行しましょう。
