displayプロパティ

要素の表示方法を変えるのが display プロパティです。

/* block: 縦に積み重なる（div, p, h1 などのデフォルト）*/ .box { display: block; } /* inline: 横に並ぶ（a, span などのデフォルト）*/ .link { display: inline; } /* inline-block: 横に並びつつwidthやheightを設定できる */ .icon { display: inline-block; } /* flex: フレックスコンテナ（横並びレイアウトの基本）*/ .row { display: flex; } /* none: 非表示にする */ .hidden { display: none; }

Flexbox

Flexboxは横並びや縦並びのレイアウトを簡単に作れます。親要素に display: flex を指定します。

基本プロパティ（親要素）

.container { display: flex; /* 並ぶ方向 */ flex-direction: row; /* 横並び（デフォルト）*/ flex-direction: column; /* 縦並び */ /* 主軸方向の配置 */ justify-content: flex-start; /* 左寄せ */ justify-content: center; /* 中央揃え */ justify-content: flex-end; /* 右寄せ */ justify-content: space-between; /* 両端揃え */ justify-content: space-around; /* 均等配置 */ /* 交差軸方向の配置 */ align-items: stretch; /* 引き伸ばす（デフォルト）*/ align-items: center; /* 中央揃え */ align-items: flex-start; /* 上揃え */ /* 隙間 */ gap: 20px; }

基本プロパティ（子要素）

.item { flex: 1; /* 残りスペースを均等に分配 */ flex-grow: 2; /* 他の要素の2倍の幅に */ }
Flexboxの覚え方
justify-content = 横軸（主軸）の配置
align-items = 縦軸（交差軸）の配置
flex-direction: column にすると、justify-content と align-items の軸が入れ替わります。

positionプロパティ

/* static: デフォルト（普通の位置）*/ .normal { position: static; } /* relative: 自分の元の位置を基準にずらす */ .shifted { position: relative; top: 10px; left: 20px; } /* absolute: 親要素（position非staticな要素）を基準に配置 */ .overlay { position: absolute; top: 0; right: 0; } /* fixed: ブラウザウィンドウを基準に固定（スクロールしても動かない）*/ .sticky-header { position: fixed; top: 0; width: 100%; }

レスポンシブ: メディアクエリ

画面幅に応じてスタイルを変えられます。

/* 基本スタイル（全画面サイズに適用）*/ .container { display: flex; } /* 768px以下の場合（スマートフォン）*/ @media (max-width: 768px) { .container { flex-direction: column; /* 縦並びに変更 */ } }

練習問題

問1: flexboxで3つのボックスを横並び

display: flex を使って、3つのdivを横並びにしてください。

各ボックスに背景色と100px以上の高さをつけましょう。


問2: justify-content: space-between

3つのボックスを justify-content: space-between で均等配置してください。


問3: 中央揃えレイアウト

flexboxを使ってコンテンツを縦横中央揃えにしてください。

  • 親要素に height: 200px を設定
  • justify-content: center と align-items: center を使う

問4: ナビゲーションバーをflexboxで作る

flexboxを使ってナビゲーションバーを作ってください。

  • 左にロゴ、右にリンク群を配置（space-between）
  • 背景色・文字色・paddingを設定

問5: 2カラムレイアウト

ヘッダー・メインコンテンツ・フッターを持つ2カラムレイアウトを作ってください。

  • ヘッダーとフッターは横幅100%
  • メインコンテンツはサイドバー（左30%）と本文（右70%）に分割
