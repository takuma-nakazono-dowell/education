CSSとは

CSS（Cascading Style Sheets）はHTMLの見た目を定義する言語です。色・フォント・余白・レイアウトなどを指定できます。

CSSの書き方 3種類

/* 1. インライン: タグに直接書く（優先度が高いが管理しにくい）*/ <p style="color: red; font-size: 20px;">テキスト</p> /* 2. styleタグ: HTML内のheadに書く */ <style> p { color: blue; } </style> /* 3. 外部ファイル: .cssファイルを読み込む（最も推奨）*/ <link rel="stylesheet" href="style.css">

セレクタの種類

/* 要素セレクタ: タグ名で指定 */ p { color: black; } /* クラスセレクタ: .クラス名 */ .highlight { background: yellow; } /* IDセレクタ: #ID名（ページ内で1つ）*/ #header { background: navy; } /* 子孫セレクタ: 半角スペースで区切る */ .card p { font-size: 14px; } /* 複数指定: カンマで区切る */ h1, h2 { color: #333; }

テキスト関連プロパティ

p { color: #333; /* 文字色 */ font-size: 16px; /* 文字サイズ */ font-weight: bold; /* 太字 */ font-family: 'Meiryo'; /* フォント */ text-align: center; /* 中央揃え */ line-height: 1.8; /* 行間 */ text-decoration: none; /* 下線なし */ }

背景プロパティ

div { background-color: #e3f2fd; /* 背景色 */ background-image: url('bg.jpg'); /* 背景画像 */ background-size: cover; /* 背景サイズ */ }

ボックスモデル

すべての要素はボックス（箱）として扱われます。中から順に content → padding → border → margin です。

.box { width: 200px; /* コンテンツの幅 */ height: 100px; /* コンテンツの高さ */ padding: 15px; /* 内側の余白 */ border: 2px solid blue; /* 境界線 */ margin: 20px; /* 外側の余白 */ } /* borderの個別指定 */ .box { border-width: 2px; border-style: solid; /* solid/dashed/dotted */ border-color: #2196F3; border-radius: 8px; /* 角を丸める */ }
box-sizing: border-box
通常、paddingとborderはwidthに加算されます。
box-sizing: border-box を設定すると、widthにpadding/borderを含めて計算できます。

練習問題

問1: styleタグでpタグをスタイリング

styleタグを使って、pタグにcolor（赤）とfont-size（20px）を設定してください。


問2: クラスセレクタで2種類のスタイル

2つの異なるクラスを定義して、それぞれ別のpタグに適用してください。

例: .info（青背景）と .warning（赤背景）


問3: ボックスにpadding・border・margin

divにpadding・border・marginを設定して、箱らしいデザインにしてください。


問4: カードデザインを作る

背景色と文字色を組み合わせてカード風のデザインを作ってください。

  • 角丸（border-radius）
  • 影（box-shadow: 0 2px 8px rgba(0,0,0,0.1)）
  • padding で内側に余白

問5: スタイリングしたページを作る

見出し・段落・ボタンをCSSでスタイリングした小さなページを作ってください。

  • h1に色とborderを設定
  • pにfont-sizeとline-heightを設定
  • buttonに背景色・文字色・padding・border-radiusを設定
