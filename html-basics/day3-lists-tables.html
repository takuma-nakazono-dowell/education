リスト

順序なしリスト: ul

順番に意味のないリストには <ul> を使います。各項目は <li> で囲みます。

<ul> <li>りんご</li> <li>バナナ</li> <li>みかん</li> </ul>

順序ありリスト: ol

手順や順番に意味があるリストには <ol> を使います。自動的に番号がつきます。

<ol> <li>水を沸騰させる</li> <li>パスタを入れる</li> <li>8分ゆでる</li> </ol>

定義リスト: dl

用語と説明のペアには <dl> を使います。

<dl> <dt>HTML</dt> <dd>ウェブページの構造を定義する言語</dd> <dt>CSS</dt> <dd>ウェブページのデザインを定義する言語</dd> </dl>

ネストしたリスト

リストの中にリストを入れることができます。

<ul> <li>果物 <ul> <li>りんご</li> <li>バナナ</li> </ul> </li> <li>野菜</li> </ul>

テーブル

テーブルの基本構造

テーブルは <table> で作ります。行は <tr>、ヘッダーは <th>、データは <td> です。

<table> <!-- ヘッダー行 --> <tr> <th>名前</th> <th>年齢</th> </tr> <!-- データ行 --> <tr> <td>田中</td> <td>25</td> </tr> </table>

thead・tbody・tfoot

テーブルをセクションに分けることで、構造が明確になります。

<table> <thead> <tr><th>商品</th><th>価格</th></tr> </thead> <tbody> <tr><td>りんご</td><td>150円</td></tr> <tr><td>バナナ</td><td>100円</td></tr> </tbody> <tfoot> <tr><td>合計</td><td>250円</td></tr> </tfoot> </table>

colspan・rowspan

セルを複数列・複数行にまたがって結合できます。

<!-- colspan: 横方向に結合（2列分）--> <td colspan="2">2列分のセル</td> <!-- rowspan: 縦方向に結合（2行分）--> <td rowspan="2">2行分のセル</td>
よくあるミス
・trの中にtdではなくliを書いてしまう
・colspanで結合した分のtdを書き忘れる

練習問題

問1: 買い物リストをulで作る

ulとliを使って、買い物リストを5項目作ってください。


問2: 手順リストをolで作る

olとliを使って、調理手順などのリストを作ってください。

例: カップラーメンの作り方、パスタのゆで方 など


問3: ネストしたリストを作る

大カテゴリと小カテゴリを持つリストを作ってください。

例: 「食べ物」の下に「果物」「野菜」、それぞれの下に具体的な品目


問4: 3列4行のテーブルを作る

tableタグを使って、以下の構成のテーブルを作ってください。

  • 列: 商品名・価格・在庫
  • 行: ヘッダー1行 + データ3行

問5: theadとtbodyに分けたテーブル

theadとtbodyを使って、テーブルを構造的に書いてください。
