集計関数

複数の行から1つの結果を計算します。

関数意味例
COUNT件数COUNT(*)
SUM合計SUM(price)
AVG平均AVG(price)
MAX最大値MAX(price)
MIN最小値MIN(price)
-- 商品の件数 SELECT COUNT(*) FROM products; -- 価格の合計、平均、最大、最小 SELECT SUM(price) AS 合計, AVG(price) AS 平均, MAX(price) AS 最大, MIN(price) AS 最小 FROM products;

GROUP BY: グループ化

指定した列でグループを作り、グループごとに集計します。

-- カテゴリごとの商品数 SELECT category_id, COUNT(*) AS 商品数 FROM products GROUP BY category_id;
ポイント: GROUP BYのルール
SELECTに書ける列は
・GROUP BYで指定した列
・集計関数（COUNT, SUM等）
のどちらかです。

複数の集計

-- カテゴリごとの商品数と平均価格 SELECT category_id, COUNT(*) AS 商品数, AVG(price) AS 平均価格, SUM(stock) AS 在庫合計 FROM products GROUP BY category_id;

HAVING: 集計結果の絞り込み

GROUP BYの結果に条件をつけます。

-- 平均価格が150以上のカテゴリ SELECT category_id, AVG(price) AS 平均価格 FROM products GROUP BY category_id HAVING AVG(price) >= 150;
WHERE と HAVING の違い
・WHERE: グループ化の前に絞り込む
・HAVING: グループ化の後に絞り込む
-- WHEREとHAVINGを両方使う例 SELECT category_id, COUNT(*) AS cnt FROM products WHERE stock >= 100 -- 先に絞り込み GROUP BY category_id HAVING COUNT(*) >= 2; -- 集計結果で絞り込み
products: id, name, category_id, price, stock
orders: id, product_id, quantity, order_date
