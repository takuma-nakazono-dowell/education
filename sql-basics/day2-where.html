WHERE句とは？

WHERE句を使うと、条件に合うデータだけを取得できます。

基本構文

SELECT 列名 FROM テーブル名 WHERE 条件;

比較演算子

演算子意味例
=等しいprice = 100
<> または !=等しくないprice <> 100
>より大きいprice > 100
<より小さいprice < 100
>=以上price >= 100
<=以下price <= 100
-- 価格が150以上の商品 SELECT * FROM products WHERE price >= 150;

AND と OR

-- 価格100以上 かつ 在庫100以上 SELECT * FROM products WHERE price >= 100 AND stock >= 100; -- カテゴリが1 または 2 SELECT * FROM products WHERE category_id = 1 OR category_id = 2;

IN: 複数の値のいずれか

-- カテゴリが1, 2のいずれか SELECT * FROM products WHERE category_id IN (1, 2);

BETWEEN: 範囲指定

-- 価格が100〜200の範囲 SELECT * FROM products WHERE price BETWEEN 100 AND 200;

LIKE: 部分一致

-- 名前に「ん」を含む SELECT * FROM products WHERE name LIKE '%ん%'; -- 「り」で始まる SELECT * FROM products WHERE name LIKE 'り%';
LIKE のワイルドカード
% : 任意の文字列（0文字以上）
_ : 任意の1文字

IS NULL / IS NOT NULL

-- NULLのデータを検索 SELECT * FROM products WHERE name IS NULL; -- NULLでないデータ SELECT * FROM products WHERE name IS NOT NULL;
products: id, name, category_id, price, stock

練習問題

問1: 価格で絞り込み

価格が 200円以上 の商品を取得してください。


問2: 等価比較

category_id が 2（飲料）の商品を取得してください。


問3: AND条件

category_id が 1（食品）で、かつ価格が 120円以下の商品を取得してください。


問4: IN を使う

id が 1, 3, 5, 7 のいずれかの商品を取得してください。


問5: BETWEEN を使う

在庫数（stock）が 80〜150 の範囲の商品を取得してください。


問6: LIKE を使う

商品名に「ん」を含む商品を取得してください。


問7: 複合条件

以下の条件を満たす商品を取得してください：

  • category_id が 1 または 2
  • かつ、在庫が 100 以上

ヒント: (条件1 OR 条件2) AND 条件3 のように括弧を使う
