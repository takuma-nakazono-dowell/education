SELECT文とは？

SELECT文は、データベースからデータを取得するための命令です。SQLの中で最もよく使います。

基本構文

SELECT 列名1, 列名2, ... FROM テーブル名;

全ての列を取得: *

-- products テーブルの全列を取得 SELECT * FROM products;
ポイント: * は「全て」の意味
アスタリスク（*）を使うと、テーブルの全ての列を取得できます。

特定の列だけ取得

-- 商品名と価格だけ取得 SELECT name, price FROM products;

列に別名をつける: AS

-- 列名を日本語にして表示 SELECT name AS 商品名, price AS 価格 FROM products;

計算結果を表示

-- 税込価格を計算 SELECT name, price, price * 1.1 AS 税込価格 FROM products;

重複を除く: DISTINCT

-- カテゴリIDの種類を取得（重複なし） SELECT DISTINCT category_id FROM products;

使用するテーブル

products: id, name, category_id, price, stock
データベースを準備中...

練習問題

問1: 全件取得

products テーブルの全データを取得してください。


問2: 特定の列を取得

products テーブルから「商品名（name）」と「在庫数（stock）」だけを取得してください。


問3: 列に別名をつける

products テーブルから name を「商品名」、price を「価格」という別名で取得してください。


問4: 計算結果を表示

商品名、価格、税込価格（price × 1.1）を取得してください。
税込価格には「tax_included」という別名をつけてください。


問5: 重複を除く

products テーブルに含まれる category_id の種類を重複なしで取得してください。


問6: 在庫金額の計算

各商品の「商品名」「価格」「在庫数」「在庫金額（price × stock）」を取得してください。
在庫金額には「total_value」という別名をつけてください。
