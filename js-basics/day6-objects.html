オブジェクトとは？

オブジェクトは、関連するデータをまとめて管理できる仕組みです。
「キー」と「値」のペアで構成されます。

person = {
  name: "田中太郎",  ← キー: 値
  age: 25,
  city: "東京"
}

オブジェクトの作成

const person = { name: "田中太郎", age: 25, city: "東京" }; console.log(person);

プロパティへのアクセス

const person = { name: "田中太郎", age: 25 }; // ドット記法（よく使う） console.log(person.name); // 田中太郎 console.log(person.age); // 25 // ブラケット記法（変数を使う時に便利） console.log(person["name"]); // 田中太郎 const key = "age"; console.log(person[key]); // 25

プロパティの変更・追加

const person = { name: "田中太郎", age: 25 }; // 変更 person.age = 26; // 追加 person.city = "東京"; console.log(person); // { name: "田中太郎", age: 26, city: "東京" }

プロパティの削除

const person = { name: "田中太郎", age: 25, city: "東京" }; delete person.city; console.log(person); // { name: "田中太郎", age: 25 }

オブジェクトの配列

実務では「オブジェクトの配列」をよく使います。

const products = [ { name: "りんご", price: 150 }, { name: "バナナ", price: 100 }, { name: "みかん", price: 80 } ]; // 2番目の商品の名前 console.log(products[1].name); // バナナ // 全商品の名前を出力 products.forEach(product => { console.log(product.name); });
Vue.jsでよく見る形
APIからデータを取得すると、大抵このような
「オブジェクトの配列」形式で返ってきます。

オブジェクトの配列を操作

const products = [ { name: "りんご", price: 150, stock: 10 }, { name: "バナナ", price: 100, stock: 0 }, { name: "みかん", price: 80, stock: 5 } ]; // 在庫がある商品だけ抽出 const inStock = products.filter(p => p.stock > 0); // 商品名だけの配列を作る const names = products.map(p => p.name); // 合計金額を計算（在庫 x 価格） const total = products.reduce((sum, p) => sum + p.price * p.stock, 0);

分割代入

オブジェクトのプロパティを変数に取り出す便利な書き方です。

const person = { name: "田中太郎", age: 25, city: "東京" }; // 従来の書き方 const name1 = person.name; const age1 = person.age; // 分割代入（おすすめ） const { name, age, city } = person; console.log(name); // 田中太郎 console.log(age); // 25

スプレッド構文

オブジェクトをコピーしたり、一部を変更した新しいオブジェクトを作れます。

const person = { name: "田中", age: 25 }; // コピー const copy = { ...person }; // 一部を変更した新しいオブジェクト const updated = { ...person, age: 26 }; console.log(updated); // { name: "田中", age: 26 }
