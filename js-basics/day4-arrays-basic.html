配列とは？

配列は、複数の値をまとめて管理できる「箱の列」のようなものです。

fruits = [ "りんご", "バナナ", "みかん" ]
           ↑       ↑       ↑
インデックス  0       1       2

配列の作成

// 配列の作成 const fruits = ["りんご", "バナナ", "みかん"]; const numbers = [1, 2, 3, 4, 5]; const mixed = ["太郎", 25, true]; // 異なる型も可 // 空の配列 const empty = [];

要素へのアクセス

const fruits = ["りんご", "バナナ", "みかん"]; console.log(fruits[0]); // りんご（最初） console.log(fruits[1]); // バナナ（2番目） console.log(fruits[2]); // みかん（3番目）
重要: インデックスは0から始まる！
最初の要素は [0]、2番目は [1]、3番目は [2] です。
これはプログラミングでは一般的なルールです。

配列の長さ: length

const fruits = ["りんご", "バナナ", "みかん"]; console.log(fruits.length); // 3 // 最後の要素を取得 console.log(fruits[fruits.length - 1]); // みかん

要素の変更

const fruits = ["りんご", "バナナ", "みかん"]; fruits[1] = "ぶどう"; // バナナをぶどうに変更 console.log(fruits); // ["りんご", "ぶどう", "みかん"]
const でも中身は変えられる
const は「変数への再代入」を禁止するだけ。
配列の中身を変更することは可能です。

要素の追加: push

const fruits = ["りんご", "バナナ"]; fruits.push("みかん"); // 末尾に追加 console.log(fruits); // ["りんご", "バナナ", "みかん"]

要素の削除: pop, shift

const fruits = ["りんご", "バナナ", "みかん"]; // 末尾を削除 const last = fruits.pop(); console.log(last); // みかん console.log(fruits); // ["りんご", "バナナ"] // 先頭を削除 const first = fruits.shift(); console.log(first); // りんご console.log(fruits); // ["バナナ"]

配列とループ

const fruits = ["りんご", "バナナ", "みかん"]; // for文で全要素を処理 for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); } // for...of（シンプルに書ける） for (const fruit of fruits) { console.log(fruit); }

配列の検索: includes, indexOf

const fruits = ["りんご", "バナナ", "みかん"]; // 含まれているか（true/false） console.log(fruits.includes("バナナ")); // true console.log(fruits.includes("ぶどう")); // false // 位置を取得（なければ-1） console.log(fruits.indexOf("バナナ")); // 1 console.log(fruits.indexOf("ぶどう")); // -1
