演算子

算術演算子

演算子意味例
+足し算5 + 3 = 8
-引き算5 - 3 = 2
*掛け算5 * 3 = 15
/割り算6 / 3 = 2
%余り7 % 3 = 1
const a = 10; const b = 3; console.log(a + b); // 13 console.log(a - b); // 7 console.log(a * b); // 30 console.log(a / b); // 3.333... console.log(a % b); // 1（余り）

比較演算子

演算子意味例
===等しい5 === 5 → true
!==等しくない5 !== 3 → true
>より大きい5 > 3 → true
<より小さい5 < 3 → false
>=以上5 >= 5 → true
<=以下5 <= 3 → false
ポイント: === と == の違い
=== は型も含めて比較（厳密等価）
== は型を変換して比較（曖昧）
基本的に === を使いましょう。

論理演算子

演算子意味例
&&かつ（AND）true && false → false
||または（OR）true || false → true
!否定（NOT）!true → false

条件分岐: if文

基本形

const age = 20; if (age >= 20) { console.log("成人です"); }

if-else

const age = 15; if (age >= 20) { console.log("成人です"); } else { console.log("未成年です"); }

if-else if-else

const score = 75; if (score >= 90) { console.log("A"); } else if (score >= 70) { console.log("B"); } else if (score >= 50) { console.log("C"); } else { console.log("D"); }

三項演算子

シンプルな条件分岐を1行で書けます。

const age = 20; // 条件 ? trueの時の値 : falseの時の値 const status = age >= 20 ? "成人" : "未成年"; console.log(status); // 成人

複数条件の組み合わせ

const age = 25; const hasLicense = true; // AND: 両方満たす場合 if (age >= 18 && hasLicense) { console.log("運転できます"); } // OR: どちらか満たす場合 const isWeekend = true; const isHoliday = false; if (isWeekend || isHoliday) { console.log("休みです"); }

出力例: 入場料は 1000 円です
