関数とは？

関数は、処理をまとめて名前をつけたものです。
何度も使う処理を関数にしておくと、コードがスッキリします。

function 挨拶する(名前) {
  「こんにちは、{名前}さん」と表示
}

挨拶する("田中");  ← 呼び出し
→ こんにちは、田中さん

関数の定義と呼び出し

// 関数の定義 function greet() { console.log("こんにちは！"); } // 関数の呼び出し greet(); // こんにちは！ greet(); // 何度でも呼べる

引数（パラメータ）

関数に値を渡すことができます。

function greet(name) { console.log(`こんにちは、${name}さん！`); } greet("田中"); // こんにちは、田中さん！ greet("鈴木"); // こんにちは、鈴木さん！

複数の引数

function introduce(name, age) { console.log(`${name}さんは${age}歳です`); } introduce("田中", 25); // 田中さんは25歳です

戻り値（return）

関数から値を返すことができます。

function add(a, b) { return a + b; } const result = add(3, 5); console.log(result); // 8 // 直接使うこともできる console.log(add(10, 20)); // 30
return の後は実行されない
return が実行されると、そこで関数が終了します。

デフォルト引数

function greet(name = "ゲスト") { console.log(`こんにちは、${name}さん！`); } greet("田中"); // こんにちは、田中さん！ greet(); // こんにちは、ゲストさん！

アロー関数

関数をより短く書ける構文です。現代のJavaScriptでよく使います。

// 従来の書き方 function add(a, b) { return a + b; } // アロー関数 const add = (a, b) => { return a + b; }; // さらに短く（1行で return できる場合） const add = (a, b) => a + b; // 引数が1つの場合、括弧も省略可 const double = n => n * 2;

配列メソッドとの組み合わせ

const numbers = [1, 2, 3, 4, 5]; // アロー関数を使う const doubled = numbers.map(n => n * 2); const evens = numbers.filter(n => n % 2 === 0); const sum = numbers.reduce((acc, n) => acc + n, 0);

関数を変数に入れる

// 関数を変数に代入 const greet = function(name) { console.log(`こんにちは、${name}さん！`); }; // アロー関数で書くとこう const greet = (name) => { console.log(`こんにちは、${name}さん！`); }; greet("田中");

オブジェクトを返す

// オブジェクトを返す場合は () で囲む const createUser = (name, age) => ({ name: name, age: age }); // 省略記法（キーと変数名が同じ場合） const createUser = (name, age) => ({ name, age }); const user = createUser("田中", 25); console.log(user); // { name: "田中", age: 25 }
