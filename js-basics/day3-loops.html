ループとは？

同じ処理を繰り返し実行するための仕組みです。

for文の基本

// for (初期化; 条件; 更新) { 処理 } for (let i = 0; i < 5; i++) { console.log(i); } // 0, 1, 2, 3, 4 が順番に出力される
for文の3つの要素
1. let i = 0 : カウンター変数の初期化
2. i < 5 : 繰り返し条件（trueの間続く）
3. i++ : 各ループ後の更新（i = i + 1 と同じ）

1から10まで出力

for (let i = 1; i <= 10; i++) { console.log(i); } // 1, 2, 3, ... 10

逆順ループ

for (let i = 5; i >= 1; i--) { console.log(i); } // 5, 4, 3, 2, 1

2ずつ増やす

for (let i = 0; i <= 10; i += 2) { console.log(i); } // 0, 2, 4, 6, 8, 10

while文

条件がtrueの間、繰り返します。

let count = 0; while (count < 5) { console.log(count); count++; // これを忘れると無限ループ！ } // 0, 1, 2, 3, 4
注意: 無限ループ
while文で条件が常にtrueだと永遠に止まりません。
必ずループを抜ける条件を用意してください。

break と continue

break: ループを抜ける

for (let i = 1; i <= 10; i++) { if (i === 5) { break; // 5になったらループ終了 } console.log(i); } // 1, 2, 3, 4

continue: 次のループへスキップ

for (let i = 1; i <= 5; i++) { if (i === 3) { continue; // 3をスキップ } console.log(i); } // 1, 2, 4, 5 (3がない)

ループの活用例

合計を求める

let sum = 0; for (let i = 1; i <= 10; i++) { sum += i; // sum = sum + i } console.log(sum); // 55

掛け算の表

const n = 3; for (let i = 1; i <= 9; i++) { console.log(`${n} x ${i} = ${n * i}`); } // 3 x 1 = 3 // 3 x 2 = 6 // ...

練習問題

問1: 1から10まで出力

for文を使って、1から10までの数字を出力してください。


問2: 合計を計算

1から100までの合計を計算して出力してください。

期待する出力: 合計: 5050


問3: 偶数だけ出力

1から20までの偶数だけを出力してください。

方法1: 2ずつ増やす / 方法2: continueを使う


問4: 九九の表（5の段）

5の段の九九を出力してください。

出力形式: 5 x 1 = 5


問5: カウントダウン

10から1までカウントダウンして、最後に「発射！」と出力してください。


問6: 特定の数で停止

1から順に足していき、合計が100を超えたらループを止めてください。

何を足した時に100を超えたか、その時の合計はいくつかを出力してください。
