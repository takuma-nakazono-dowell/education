配列メソッドとは？

配列には便利なメソッド（機能）が用意されています。
ループを書かなくても、配列を効率的に操作できます。

forEach: 各要素に処理を実行

const fruits = ["りんご", "バナナ", "みかん"]; fruits.forEach(function(fruit) { console.log(fruit); }); // アロー関数で短く書ける fruits.forEach(fruit => console.log(fruit));
forEach: 配列の各要素に対して関数を実行
[りんご, バナナ, みかん].forEach(出力)
→ りんご を出力
→ バナナ を出力
→ みかん を出力

map: 各要素を変換して新しい配列を作る

const numbers = [1, 2, 3, 4, 5]; // 各要素を2倍にした新しい配列を作る const doubled = numbers.map(num => num * 2); console.log(doubled); // [2, 4, 6, 8, 10] console.log(numbers); // [1, 2, 3, 4, 5]（元は変わらない）
map: 各要素を変換して新しい配列を返す
[1, 2, 3].map(x => x * 2)
→ [2, 4, 6]
ポイント: map は新しい配列を返す
元の配列は変更されません。
結果を変数に入れて使います。

filter: 条件に合う要素だけ抽出

const numbers = [1, 2, 3, 4, 5, 6]; // 偶数だけ抽出 const evens = numbers.filter(num => num % 2 === 0); console.log(evens); // [2, 4, 6]
filter: 条件に合う要素だけの配列を返す
[1, 2, 3, 4, 5, 6].filter(偶数?)
→ [2, 4, 6]

find: 条件に合う最初の要素を返す

const numbers = [10, 20, 30, 40]; // 25より大きい最初の要素 const found = numbers.find(num => num > 25); console.log(found); // 30

some / every: 条件チェック

const numbers = [1, 2, 3, 4, 5]; // some: 1つでも条件を満たすか console.log(numbers.some(n => n > 3)); // true // every: 全てが条件を満たすか console.log(numbers.every(n => n > 0)); // true console.log(numbers.every(n => n > 3)); // false

reduce: 配列を1つの値にまとめる

const numbers = [1, 2, 3, 4, 5]; // 合計を求める const sum = numbers.reduce((total, num) => total + num, 0); console.log(sum); // 15
reduce: 配列を1つの値にまとめる
[1, 2, 3, 4, 5].reduce(足す, 初期値0)
→ 0 + 1 = 1
→ 1 + 2 = 3
→ 3 + 3 = 6
→ 6 + 4 = 10
→ 10 + 5 = 15

メソッドチェーン

メソッドを繋げて書くことができます。

const numbers = [1, 2, 3, 4, 5, 6]; // 偶数だけ取り出して、2倍にする const result = numbers .filter(n => n % 2 === 0) // [2, 4, 6] .map(n => n * 2); // [4, 8, 12] console.log(result); // [4, 8, 12]
