🖼️ Vue.js 応用学習

コンポーネント設計・ライフサイクル・非同期処理

前提知識：Vue.js 基礎（データバインディング・イベント・コンポーネント基礎・Computed）を完了していること

この教材について

Vue.js 基礎で学んだ props / $emit を土台に、実務でよく使う3つのトピックを学びます。

  • emits宣言・v-model・provide/inject：コンポーネント間でデータを受け渡す発展的な方法
  • ライフサイクルフック：コンポーネントの「生まれてから消えるまで」を制御する
  • 非同期処理連携：サーバーからデータを取得してVueに表示する

カリキュラム

Day 1
コンポーネント間通信（発展）
emits宣言 / v-model on コンポーネント / provide・inject
  • emitsオプションで発火するイベントを明示する
  • v-modelをカスタムコンポーネントに対応させる
  • provide/injectで深い階層にデータを渡す
Day 2
ライフサイクルフック
created / mounted / updated / unmounted
  • コンポーネントの「一生」と各フックの役割
  • created：初期データ取得・初期化処理
  • mounted：DOM操作・外部ライブラリ初期化
  • updated：再レンダリング後の処理
  • unmounted：タイマー・イベントリスナーのクリーンアップ
Day 3
非同期処理連携
fetch API / ローディング状態 / エラーハンドリング
  • なぜVueで非同期が必要か
  • mounted + async/await でデータ取得
  • isLoading フラグでローディング表示
  • try/catch でエラーを画面に表示
