なぜVueで非同期が必要か？

実務のWebアプリでは、サーバーからデータを取得して画面に表示する処理が必ず発生します。 サーバーとの通信は時間がかかるため、非同期処理（async/await）を使います。

非同期なしだとどうなる？
通信が終わるまでブラウザ全体がフリーズします。
async/await を使うことで「通信中も他の処理を続けられる」状態を作ります。

Vueでの非同期パターン

最も一般的なパターン：mounted で API を呼ぶ

Vue.createApp({ data() { return { users: [], // 取得したデータを入れる配列 isLoading: false, // 読み込み中フラグ error: null // エラーメッセージ } }, async mounted() { // mounted を async にする this.isLoading = true try { const res = await fetch('/api/users') this.users = await res.json() } catch (e) { this.error = 'データの取得に失敗しました' } finally { this.isLoading = false // 成功・失敗どちらでも必ず解除 } } })

テンプレート側の3状態を表示

<!-- ① 読み込み中 --> <div v-if="isLoading">読み込み中...</div> <!-- ② エラー --> <div v-else-if="error">{{ error }}</div> <!-- ③ 正常表示 --> <ul v-else> <li v-for="user in users" :key="user.id"> {{ user.name }} </li> </ul>
3状態を必ず用意する
① ローディング中 → スピナーやスケルトン表示
② エラー → ユーザーに伝わるメッセージ表示
③ 正常 → データ表示
finally で isLoading を必ず false に戻すことを忘れずに。

メソッドに切り出す

mounted が長くなる場合はメソッドに切り出すのが一般的です：

{ async mounted() { await this.fetchUsers() // メソッドを呼ぶだけ }, methods: { async fetchUsers() { this.isLoading = true try { const res = await fetch('/api/users') this.users = await res.json() } catch (e) { this.error = e.message } finally { this.isLoading = false } } } }

動くデモ & 練習問題

デモ 1: mounted でユーザー一覧を取得

画面が表示されると同時に API を呼び出してユーザー一覧を取得します。
mounted + async/await + fetch の基本パターンです。

デモ 2: ボタンでデータを再取得

ボタンをクリックすると投稿データを取得します。
取得中はボタンが無効化されローディング表示になります。
{{ post.title }}
{{ post.body }}

デモ 3: エラーハンドリング

「正常取得」は成功します。「存在しないURL」はエラーになります。
try/catch でエラーをキャッチして画面に表示します。
取得中...
エラー: {{ errMsg }}
取得成功: {{ result.name }} ({{ result.email }})
ボタンを押してください

練習: ID を指定して投稿を取得

入力欄に1〜100の数字を入力して「取得」ボタンを押すと、 その ID の投稿を JSONPlaceholder から取得して表示してください。
URL: https://jsonplaceholder.typicode.com/posts/{id}
読み込み中...
{{ fetchError }}
Post #{{ fetchedPost.id }}
{{ fetchedPost.title }}

ヒント
methods に async fetchById() を定義します。
URL は `https://jsonplaceholder.typicode.com/posts/${this.inputId}`
入力値のバリデーション（1〜100かどうか）も忘れずに。
答えを見る
data() { return { inputId: '', fetchedPost: null, loading: false, fetchError: null } }, methods: { async fetchById() { const id = parseInt(this.inputId) if (!id || id < 1 || id > 100) { this.fetchError = '1〜100の数値を入力してください' return } this.loading = true this.fetchError = null try { const res = await fetch( `https://jsonplaceholder.typicode.com/posts/${id}` ) if (!res.ok) throw new Error(`HTTP ${res.status}`) this.fetchedPost = await res.json() } catch (e) { this.fetchError = e.message } finally { this.loading = false } } }
