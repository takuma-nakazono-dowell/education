素数について

素数の定義
1より大きい自然数のうち、
1とその数自身以外で割り切れない数
例: 2, 3, 5, 7, 11, 13, 17...
効率的な判定のヒント
n が素数かどうかを調べるには、
2 〜 √n まで割り切れる数があるか確認すればよい
Math.sqrt(n) で平方根を求められる
2は唯一の偶数の素数
1は素数ではない（定義より）
2は素数（2自身でしか割れない）

曜日計算について

曜日は7日で循環する
基準日の曜日がわかれば、
日数差 % 7 で目標日の曜日がわかる
グレゴリオ暦のうるう年
① 4で割り切れる → うるう年
② ただし100で割り切れる → 平年
③ ただし400で割り切れる → うるう年
例: 2000年=うるう年, 1900年=平年
月ごとの日数
1,3,5,7,8,10,12月 → 31日
4,6,9,11月 → 30日
2月 → うるう年28日、平年27日
ツェラーの公式は使わないこと
自分でロジックを組み立ててください。
ヒント: 1970年1月1日（木曜日）や
2000年1月1日（土曜日）を基準に
日数を数える方法が考えやすい。
400年 = 146097日
グレゴリオ暦は400年で完全に循環する。
146097 % 7 = 0 なので、
400年後は同じ曜日になる。
