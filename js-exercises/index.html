プログラミングの基礎ロジックをブラウザで実践

前提知識：JavaScript基礎（変数・条件・ループ・配列・関数）を完了していること

この教材について

C# 基礎研修の演習問題と同じロジック課題を、JavaScript で解きます。
コンソールアプリの代わりにブラウザ上で実行でき、テストケースで自動的に正解チェックされます。

AIは使わずに自力で考えてください。わからない場合はヒントを参照してください。

問題の解き方：各問に 関数のスケルトン が用意されています。 関数の中身を実装して「実行」ボタンを押すとテストケースで合否が判定されます。

カリキュラム

Phase
1

条件・ループ

変数・条件分岐・ループ・文字列操作・エラーハンドリング

  • 数値配列の合計を計算する
  • 加算処理を別関数（サブルーチン）に切り出す
  • 文字の種別を判定する（数字／英字／その他）
  • 種別ごとの個数をカウントする
  • 合計値の大小・奇偶を判定する
  • 数値以外の入力に対するエラーハンドリング
Phase
2

配列・アルゴリズム

配列操作・ソートアルゴリズムの実装

  • 配列の2要素を入れ替える（swap）
  • バブルソートを実装する
  • 挿入ソートを実装する
Phase
3

関数・応用ロジック

アルゴリズム設計・数学的ロジックの実装

  • 素数判定・100以下の素数一覧を求める
  • 曜日計算（自作ロジック）— 12000年1月1日は何曜日？
