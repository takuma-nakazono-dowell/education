問題の仕様

戻り値の型に注意
各問の「仕様」に記載した型で返してください。
型が違うとテストが失敗します。
例: return 15（数値）
　　return '奇数'（文字列）

問1: 数値配列の合計

数値の配列を受け取り、全要素の合計を返す関数を実装してください。
引数: numbers (number[])
戻り値: number（合計値）
（実行ボタンを押してください）

問2: 加算処理をサブルーチンに切り出す

2つの数値を足して返す add(a, b) を定義し、 それを使って配列の合計を返す calcSum2(numbers) を実装してください。
calcSum2 は add を呼び出して実装すること。
add(a, b): a + b を返す
calcSum2(numbers): add を使って合計を返す
（実行ボタンを押してください）

問3: 文字の種別を判定する

1文字（string）を受け取り、その種別を返す関数を実装してください。
・数字（0〜9）なら '数字'
・英字（a〜z, A〜Z）なら '英字'
・それ以外なら 'その他'
引数: ch (string, 1文字)
戻り値: '数字' | '英字' | 'その他'
（実行ボタンを押してください）

問4: 種別ごとの個数をカウントする

文字の配列を受け取り、「数字」「英字」「その他」の個数をオブジェクトで返す関数を実装してください。
引数: chars (string[])
戻り値: { digit: number, alpha: number, other: number }
（実行ボタンを押してください）

問5: 合計値の大小・奇偶を判定する

数値の配列を受け取り、合計値・大小判定・奇偶判定をまとめたオブジェクトを返す関数を実装してください。
引数: numbers (number[])
戻り値: {
  sum: number,
  comparison: '10以上' | '10未満',
  parity: '奇数' | '偶数'
}
（実行ボタンを押してください）

問6: エラーハンドリング

文字列の配列を受け取り、全要素を数値に変換して合計を返す関数を実装してください。
数値に変換できない要素が含まれていた場合は Error をスローすること。
引数: inputs (string[])
戻り値: number（合計）
例外: 数値以外が含まれる場合 throw new Error(...)
（実行ボタンを押してください）
