アルゴリズム解説

swap（入れ替え）の基本
一時変数を使う:
const tmp = arr[i]
arr[i] = arr[j]
arr[j] = tmp
配列のコピー
元の配列を壊さないためにコピーする:
const copy = [...arr]

バブルソート

隣り合う要素を比較し、大きい方を後ろに押し出していく。

// 例: [5, 3, 1] パス1: 5と3を比較 → 交換 → [3, 5, 1] 5と1を比較 → 交換 → [3, 1, 5] パス2: 3と1を比較 → 交換 → [1, 3, 5] 3と5を比較 → そのまま 完了: [1, 3, 5]
外側のループ: n-1 回
内側のループ: n-1-i 回
隣と比較して大きければswap

挿入ソート

未ソート部分の先頭を取り出し、ソート済み部分の正しい位置に挿入する。

// 例: [5, 3, 1] i=1: key=3 3 < 5 → 5を右シフト → [5, 5, 1] 先頭に3を挿入 → [3, 5, 1] i=2: key=1 1 < 5 → 5を右シフト → [3, 5, 5] 1 < 3 → 3を右シフト → [3, 3, 5] 先頭に1を挿入 → [1, 3, 5] 完了: [1, 3, 5]
const key = arr[i]
let j = i - 1
while (j >= 0 && arr[j] > key)
  arr[j+1] = arr[j--]
arr[j+1] = key
