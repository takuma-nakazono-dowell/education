SQL 応用学習 Part 2

この教材について

SQL基礎（Part 1）を学んだ人向けの応用コースです。
実務でよく使うサブクエリ・ウィンドウ関数・トランザクションに加え、
「重いクエリを書かない」ための知識と、フロント/バックエンドの役割分担を学びます。

前提：SQL基礎（Part 1）の内容を理解していること
