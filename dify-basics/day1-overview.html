Dify — AI開発プラットフォーム

Difyはプログラミング不要でAIアプリを構築できるオープンソースのプラットフォームです。

チャットボット、RAG（文書検索AI）、業務自動化ワークフローなどを、 ブラウザ上のGUI操作だけで作れます。

なぜDifyが注目されるのか？

  • ノーコード / ローコード：GUI操作だけでAIアプリが作れる
  • 複数のLLM対応：GPT-4、Claude、Gemini、ローカルLLMなど数十のプロバイダーに対応
  • RAG内蔵：社内文書を読ませて回答させる機能が標準装備
  • ワークフロー：複数ステップの処理をビジュアルに構築
  • オープンソース：無料でセルフホスト可能（MITライセンス）
  • API提供：作ったアプリを業務システムに組み込める

Difyの立ち位置
ChatGPTの「手軽さ」とLangChainの「自由度」の中間。
プログラミングなしで本格的なAIアプリが作れるのが最大の強み。

オープンソースとは？

DifyのソースコードはGitHub上で公開されています（MITライセンス）。

  • 誰でもコードを見て、改造して、自社サーバーにデプロイできる
  • クラウド版（dify.ai）は有料プランあり、すぐ使い始められる
  • セルフホスト版はDocker Composeで構築、データが社外に出ない
セルフホスト版の注意点
自社サーバーで動かす場合、サーバーの管理・アップデートは自己責任。
まずはクラウド版（無料枠あり）で試すのがおすすめ。

Difyの主要機能

  • 🎨 Prompt IDE — プロンプトの設計・テスト・比較ができるエディタ コア機能
  • 📚 RAGパイプライン — 文書をアップロードして検索・回答に活用 コア機能
  • 🔀 ワークフロービルダー — ドラッグ&ドロップで処理フローを構築 コア機能
  • 🤖 エージェント — Function Calling / ReActでツール呼び出し コア機能
  • 🔌 Backend-as-a-Service — 全機能にREST API提供 コア機能
  • 📊 LLMOps — ログ監視・利用分析・コスト管理 コア機能
  • 👤 Human-in-the-Loop — ワークフロー途中で人間が承認・編集 v1.13 新機能
  • 📄 Summary Index — AIがチャンク要約を生成し検索精度向上 v1.12 新機能
