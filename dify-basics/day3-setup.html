Difyの始め方

Difyの利用方法は2つあります。
まずはどちらで始めるか選びましょう。

クラウド版 vs セルフホスト版

項目 クラウド版（dify.ai） セルフホスト版（Docker）
セットアップ アカウント登録だけ Docker環境が必要
費用 無料枠あり / 有料プラン サーバー費用のみ
データの場所 Dify社のサーバー 自社サーバー内
管理・運用 不要（Dify社が管理） 自社で管理・更新
LLM費用 自分のAPIキーを設定 自分のAPIキーを設定
おすすめ まず試したい人 機密データを扱う場合
おすすめ: まずクラウド版で試す
操作を覚えてから必要に応じてセルフホストに移行するのが効率的。
クラウド版の無料枠で十分に学習できます。

LLM APIキーの準備

Dify自体は「プラットフォーム」であり、AIの頭脳（LLM）は別途必要です。

  • OpenAI: GPT-4o / GPT-4 を使う場合
  • Anthropic: Claude を使う場合
  • Google: Gemini を使う場合

いずれかのAPIキーを事前に取得しておきましょう。

APIキーの管理に注意
APIキーはパスワードと同じです。
他人に共有しない、GitHubにコミットしない、定期的にローテーションする。

セルフホスト版のインストール

Dockerが使える環境で以下を実行します。

# Difyリポジトリをクローン git clone https://github.com/langgenius/dify.git # dockerディレクトリに移動 cd dify/docker # 環境変数ファイルをコピー cp .env.example .env # 起動 docker compose up -d

起動後、http://localhost/install にアクセスして初期設定を行います。

Docker版の要件
Docker / Docker Compose が必要。メモリは最低4GB推奨。
社内ネットワークの制約（プロキシ等）に注意。

セットアップ手順

クラウド版の登録手順

  1. ブラウザで https://cloud.dify.ai にアクセス
  2. 「Get Started」をクリックし、メールアドレスで登録
    （GitHubアカウントやGoogleアカウントでも可）
  3. メール認証を完了してログイン
  4. ワークスペースが自動作成され、ダッシュボードが表示される
