ループ処理

for ループ

// 基本のfor for (int i = 0; i < 5; i++) { Console.WriteLine(i); // 0,1,2,3,4 } // 逆順 for (int i = 5; i >= 1; i--) { Console.Write($"{i} "); // 5 4 3 2 1 }

while / do-while

// while int count = 0; while (count < 3) { Console.WriteLine(count); count++; } // do-while（最低1回は実行される） int num = 0; do { Console.WriteLine(num); num++; } while (num < 3);

foreach（コレクション反復）

string[] fruits = { "りんご", "バナナ", "みかん" }; foreach (string fruit in fruits) { Console.WriteLine(fruit); }
ポイント: 配列やリストを扱うなら foreach が最もシンプルです。 インデックスが必要な場合は for を使いましょう。

break / continue

// break: ループを抜ける for (int i = 0; i < 10; i++) { if (i == 5) break; Console.Write(i + " "); // 0 1 2 3 4 } // continue: 今回のイテレーションをスキップ for (int i = 0; i < 5; i++) { if (i == 2) continue; Console.Write(i + " "); // 0 1 3 4 }

ネストしたループ

// 九九の表（一部） for (int i = 1; i <= 3; i++) { for (int j = 1; j <= 3; j++) { Console.Write($"{i}×{j}={i*j} "); } Console.WriteLine(); } // 1×1=1 1×2=2 1×3=3 // 2×1=2 2×2=4 2×3=6 // 3×1=3 3×2=6 3×3=9

練習問題

問1 かんたん

以下のコードを読んで、出力結果を答えてください。
int sum = 0; for (int i = 1; i <= 5; i++) { sum += i; } Console.WriteLine(sum);
答えを見る
15 // 1+2+3+4+5 = 15 // i=1: sum=1 // i=2: sum=3 // i=3: sum=6 // i=4: sum=10 // i=5: sum=15

問2 かんたん

以下のコードを読んで、出力結果を答えてください。
for (int i = 0; i < 6; i++) { if (i % 2 == 0) continue; Console.Write(i + " "); }
答えを見る
1 3 5 // i % 2 == 0 のとき（偶数のとき）continue でスキップされる // つまり奇数の 1, 3, 5 だけ出力される

問3 ふつう

for 文を使って、1〜100 の合計を計算して出力するコードを書いてください。
答えを見る
int sum = 0; for (int i = 1; i <= 100; i++) { sum += i; } Console.WriteLine($"1〜100の合計: {sum}"); // 出力: 1〜100の合計: 5050 // 数学的公式でも求められる: n*(n+1)/2 = 100*101/2 = 5050

問4 ふつう

for ループを使って、以下のような三角形を出力するコードを書いてください。
*
**
***
****
*****
答えを見る
// 外側のループで行数を制御する for (int i = 1; i <= 5; i++) { // 内側のループで * を i 個出力する for (int j = 0; j < i; j++) { Console.Write("*"); } Console.WriteLine(); // 改行 } // new string() を使う方法（よりシンプル） for (int i = 1; i <= 5; i++) { Console.WriteLine(new string('*', i)); }

問5 むずかしい

FizzBuzz を実装してください。
1〜30 までループし、以下のルールで出力してください。
・3の倍数 → Fizz
・5の倍数 → Buzz
・3と5両方の倍数 → FizzBuzz
・それ以外 → 数字をそのまま出力
答えを見る
for (int i = 1; i <= 30; i++) { // 3と5の倍数を先に判定する（順番が重要） if (i % 15 == 0) { Console.WriteLine("FizzBuzz"); } else if (i % 3 == 0) { Console.WriteLine("Fizz"); } else if (i % 5 == 0) { Console.WriteLine("Buzz"); } else { Console.WriteLine(i); } } // 出力例: 1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz ...
