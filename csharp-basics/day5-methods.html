メソッドの定義と活用

メソッドの基本

// 戻り値あり static int Add(int a, int b) { return a + b; } // 戻り値なし（void） static void Greet(string name) { Console.WriteLine($"こんにちは、{name}さん！"); } // 呼び出し int result = Add(3, 5); // 8 Greet("田中");

デフォルト引数

static void Introduce(string name, int age = 0, string city = "不明") { Console.WriteLine($"{name}({age}歳) 出身: {city}"); } Introduce("田中"); // 田中(0歳) 出身: 不明 Introduce("山田", 25); // 山田(25歳) 出身: 不明 Introduce("佐藤", 30, "東京"); // 佐藤(30歳) 出身: 東京

名前付き引数

// 順番を気にせず引数を渡せる Introduce(name: "鈴木", city: "大阪", age: 28);

オーバーロード

// 型や引数の数が違えば同じ名前のメソッドを定義できる static int Multiply(int a, int b) => a * b; static double Multiply(double a, double b) => a * b; static int Multiply(int a, int b, int c) => a * b * c; Console.WriteLine(Multiply(3, 4)); // 12 Console.WriteLine(Multiply(2.5, 4.0)); // 10.0 Console.WriteLine(Multiply(2, 3, 4)); // 24

out / ref 引数

// out: メソッド内で値をセットして返す static bool TryDivide(int a, int b, out double result) { if (b == 0) { result = 0; return false; } result = (double)a / b; return true; } if (TryDivide(10, 3, out double val)) Console.WriteLine(val); // 3.3333... // ref: 参照渡し（呼び出し元の変数を直接変更） static void Double(ref int x) { x *= 2; } int n = 5; Double(ref n); Console.WriteLine(n); // 10

式形式メソッド（ラムダ形式）

// 通常の書き方 static int Square(int x) { return x * x; } // 式形式（1行で書ける） static int Square(int x) => x * x;
ポイント: C#では関数はクラスの中にメソッドとして定義します。 static メソッドはインスタンス不要で呼び出せます。 .NET 6以降のトップレベルステートメントではメソッドをファイルの末尾に書けます。

練習問題

問1 かんたん

以下のコードを読んで、出力結果を答えてください。
static int Max(int a, int b) => a > b ? a : b; Console.WriteLine(Max(10, 20)); Console.WriteLine(Max(5, 3));
答えを見る
20 // 10 と 20 を比較して大きい方 = 20 5 // 5 と 3 を比較して大きい方 = 5

問2 かんたん

空欄を埋めて、コードを完成させてください。
static ____ PrintStars(int count) { for (int i = 0; i < count; i++) Console.Write("*"); Console.WriteLine(); } PrintStars(5); // *****
答えを見る
static void PrintStars(int count) // ↑ 戻り値がない場合は void を指定する { for (int i = 0; i < count; i++) Console.Write("*"); Console.WriteLine(); } PrintStars(5); // *****

問3 ふつう

オーバーロードを使って「DoubleIt」メソッドを2つ定義してください。
・DoubleIt(int n) → 数値を2倍にして返す（例: 5 → 10）
・DoubleIt(string s) → 文字列を2回繰り返して返す（例: "Hello" → "HelloHello"）
答えを見る
// int版: 数値を2倍にする static int DoubleIt(int n) => n * 2; // string版: 文字列を2回繰り返す static string DoubleIt(string s) => s + s; // 呼び出し Console.WriteLine(DoubleIt(5)); // 10 Console.WriteLine(DoubleIt("Hello")); // HelloHello // コンパイラが引数の型を見て自動的に正しいメソッドを選択する

問4 ふつう

デフォルト引数を使った CreateMessage メソッドを定義してください。
・引数: name（必須）、greeting（省略可、デフォルト「こんにちは」）
・出力例: 「こんにちは、田中さん！」「おはよう、山田さん！」
答えを見る
// デフォルト引数を定義する static string CreateMessage(string name, string greeting = "こんにちは") { return $"{greeting}、{name}さん！"; } // 呼び出し Console.WriteLine(CreateMessage("田中")); // こんにちは、田中さん！ Console.WriteLine(CreateMessage("山田", "おはよう")); // おはよう、山田さん！ Console.WriteLine(CreateMessage("佐藤", "さようなら")); // さようなら、佐藤さん！

問5 むずかしい

out 引数を使って「SumAndProduct」メソッドを定義してください。
・引数: int a, int b（入力）、out int sum, out int product（出力）
・合計と積を out 引数でそれぞれ返す
・呼び出して結果を出力してください
答えを見る
// out 引数で2つの値を返すメソッド static void SumAndProduct(int a, int b, out int sum, out int product) { sum = a + b; // out 引数はメソッド内で必ず代入する product = a * b; } // 呼び出し: out 変数はここで宣言できる（インライン宣言） SumAndProduct(4, 6, out int s, out int p); Console.WriteLine($"4 + 6 = {s}"); // 4 + 6 = 10 Console.WriteLine($"4 × 6 = {p}"); // 4 × 6 = 24 // out は複数の値を返したい場合に使う // ただし現代のC#では ValueTuple や record の方が好まれる場合もある
