演算と条件分岐

算術演算子

int a = 10, b = 3; Console.WriteLine(a + b); // 13 Console.WriteLine(a - b); // 7 Console.WriteLine(a * b); // 30 Console.WriteLine(a / b); // 3（整数除算） Console.WriteLine(a % b); // 1（余り） // インクリメント・デクリメント int count = 0; count++; // 1 count--; // 0 count += 5; // 5

比較演算子・論理演算子

int x = 10; Console.WriteLine(x > 5); // True Console.WriteLine(x == 10); // True Console.WriteLine(x != 3); // True // 論理演算子 bool p = true, q = false; Console.WriteLine(p && q); // False（AND） Console.WriteLine(p || q); // True（OR） Console.WriteLine(!p); // False（NOT）

if / else if / else

int score = 75; if (score >= 90) { Console.WriteLine("優"); } else if (score >= 70) { Console.WriteLine("良"); } else if (score >= 50) { Console.WriteLine("可"); } else { Console.WriteLine("不可"); }

switch 文 / switch 式

// switch 文 string day = "月"; switch (day) { case "土": case "日": Console.WriteLine("休日"); break; default: Console.WriteLine("平日"); break; } // switch 式（C# 8.0以降、モダンな書き方） string result = day switch { "土" or "日" => "休日", _ => "平日" };

三項演算子

int age = 20; string label = age >= 18 ? "成人" : "未成年"; Console.WriteLine(label); // 成人
ポイント: switch の case には break が必要です（C++と同様）。 switch 式は C# 8.0以降の機能でより簡潔に書けます。

練習問題

問1 かんたん

以下のコードを読んで、各行の出力結果を答えてください。
Console.WriteLine(7 / 2); Console.WriteLine(7.0 / 2); Console.WriteLine(7 % 2);
答えを見る
3 // int同士の除算は小数が切り捨てられる 3.5 // 一方が double なので小数になる 1 // 7 ÷ 2 の余りは 1

問2 かんたん

以下の条件式の結果（True / False）を答えてください。
int x = 15; Console.WriteLine(x > 10 && x < 20); Console.WriteLine(x == 15 || x == 20); Console.WriteLine(!(x > 10));
答えを見る
True // 15 > 10 かつ 15 < 20 → 両方 true なので AND = true True // 15 == 15 が true → OR は片方でも true なら true False // 15 > 10 は true → NOT で反転 → false

問3 ふつう

switch文を使って、1〜7の数字を「月曜日」〜「日曜日」に変換するコードを書いてください。
（1=月曜日、7=日曜日、範囲外は「不明」）
答えを見る
int num = 3; string dayName = num switch { 1 => "月曜日", 2 => "火曜日", 3 => "水曜日", 4 => "木曜日", 5 => "金曜日", 6 => "土曜日", 7 => "日曜日", _ => "不明" }; Console.WriteLine(dayName); // 水曜日 // switch 文で書く場合 switch (num) { case 1: Console.WriteLine("月曜日"); break; case 2: Console.WriteLine("火曜日"); break; // ... 以下同様 default: Console.WriteLine("不明"); break; }

問4 ふつう

以下の if-else 文を switch 式で書き直してください。
string grade; if (score >= 90) grade = "A"; else if (score >= 80) grade = "B"; else if (score >= 70) grade = "C"; else grade = "D";
答えを見る
// switch 式 + パターンマッチング（C# 9.0以降） string grade = score switch { >= 90 => "A", >= 80 => "B", >= 70 => "C", _ => "D" }; // 上から順に評価されるので score=85 なら "B" になる

問5 むずかしい

BMI計算プログラムを書いてください。
身長(m)と体重(kg)を変数で持ち、BMIを計算して肥満度を判定・出力してください。

BMI = 体重 / (身長 × 身長)
・18.5未満 → 低体重
・25.0未満 → 普通体重
・25.0以上 → 肥満
答えを見る
double height = 1.70; // 身長(m) double weight = 65.0; // 体重(kg) // BMI を計算する double bmi = weight / (height * height); Console.WriteLine($"BMI: {bmi:F2}"); // F2 = 小数点2桁 // 肥満度を判定する string category; if (bmi < 18.5) { category = "低体重"; } else if (bmi < 25.0) { category = "普通体重"; } else { category = "肥満"; } Console.WriteLine($"判定: {category}");
