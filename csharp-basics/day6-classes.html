クラスとオブジェクト

クラスとは

オブジェクトの設計図。フィールド（データ）とメソッド（動作）をまとめたもの。

クラスの基本構造

class Person { // フィールド（データの保持） private string _name; private int _age; // コンストラクタ（インスタンス生成時に呼ばれる） public Person(string name, int age) { _name = name; _age = age; } // プロパティ（フィールドへの安全なアクセス） public string Name { get { return _name; } set { _name = value; } } // 自動実装プロパティ（短縮形） public int Age { get; set; } // メソッド public void Introduce() { Console.WriteLine($"私は{_name}、{_age}歳です。"); } } // 使い方 var person = new Person("田中", 25); person.Introduce(); Console.WriteLine(person.Name);

アクセス修飾子

public class BankAccount { private decimal _balance; // 外からアクセス不可 public string Owner { get; } // 読み取り専用 public BankAccount(string owner, decimal initial) { Owner = owner; _balance = initial; } public void Deposit(decimal amount) { if (amount > 0) _balance += amount; } public bool Withdraw(decimal amount) { if (amount > _balance) return false; _balance -= amount; return true; } public decimal GetBalance() => _balance; }
カプセル化の原則: フィールドはprivate、アクセスはプロパティやメソッド経由が原則（カプセル化）。

static メンバー

class Counter { private static int _count = 0; // 全インスタンスで共有 public Counter() { _count++; } public static int GetCount() => _count; } new Counter(); new Counter(); Console.WriteLine(Counter.GetCount()); // 2

readonly / init

class Point { public readonly double X; // コンストラクタでのみ設定可 public double Y { get; init; } // 初期化子でのみ設定可 public Point(double x, double y) { X = x; Y = y; } } var p = new Point(3.0, 4.0); // p.X = 5.0; // エラー: readonlyなので変更不可 // initプロパティはオブジェクト初期化子でのみ設定可能

練習問題

問題 1 easy

以下のコードを読んで、出力結果を答えてください。

class Dog { public string Name { get; set; } public int Age { get; set; } public Dog(string name, int age) { Name = name; Age = age; } public void Bark() => Console.WriteLine($"{Name}：ワン！"); } var dog = new Dog("ポチ", 3); dog.Bark(); Console.WriteLine($"年齢: {dog.Age}");

ポチ：ワン！ 年齢: 3

問題 3 medium

銀行口座クラスを参考に、以下の仕様で Item クラスを書いてください。

  • フィールド: _name（private）、_price（private）
  • プロパティ: Name（読み取りのみ）、Price（get/set、0以上のみ設定可）
  • メソッド: Display() → "商品名: ○○, 価格: △△円" と出力
答えを見る
class Item { private string _name; private decimal _price; // コンストラクタ public Item(string name, decimal price) { _name = name; Price = price; // setterのバリデーションを通す } // 読み取り専用プロパティ public string Name => _name; // 0以上のみ設定可能なプロパティ public decimal Price { get { return _price; } set { if (value >= 0) _price = value; } } // 表示メソッド public void Display() { Console.WriteLine($"商品名: {_name}, 価格: {_price}円"); } } // 使用例 var item = new Item("リンゴ", 150); item.Display(); // 商品名: リンゴ, 価格: 150円 item.Price = -10; // 無視される（0未満のため） Console.WriteLine(item.Price); // 150

問題 4 medium

staticメンバーを使った IdGenerator クラスを書いてください。

  • NewId() を呼ぶたびに 1, 2, 3... と連番のIDを返す
答えを見る
class IdGenerator { // staticフィールド: 全インスタンスで共有 private static int _lastId = 0; // staticメソッド: インスタンス不要で呼べる public static int NewId() { _lastId++; return _lastId; } } // 使用例 Console.WriteLine(IdGenerator.NewId()); // 1 Console.WriteLine(IdGenerator.NewId()); // 2 Console.WriteLine(IdGenerator.NewId()); // 3

問題 5 hard

カプセル化を意識して Student クラスを設計してください。

  • 名前（private）・成績リスト（List<int>、private）を持つ
  • Name プロパティ（読み取りのみ）
  • AddScore(int score): 0〜100の範囲の点数を追加する
  • Average(): 平均点を double で返す（点数がない場合は 0 を返す）
答えを見る
class Student { private string _name; private List<int> _scores; public Student(string name) { _name = name; _scores = new List<int>(); } // 読み取り専用プロパティ public string Name => _name; // 0〜100の範囲のみ追加 public void AddScore(int score) { if (score >= 0 && score <= 100) _scores.Add(score); } // 平均点を返す（点数がなければ0） public double Average() { if (_scores.Count == 0) return 0; return _scores.Average(); } } // 使用例 var student = new Student("田中"); student.AddScore(85); student.AddScore(92); student.AddScore(78); student.AddScore(110); // 無視される（範囲外） Console.WriteLine($"{student.Name} の平均点: {student.Average():F1}点"); // 田中 の平均点: 85.0点
