C# とは？

C#はMicrosoftが開発した静的型付けのオブジェクト指向言語です。 Webアプリ・デスクトップアプリ・ゲーム開発（Unity）など幅広く使われています。 各日の教材にはコード例と練習問題があります。 Visual Studio Code + C# Dev Kit、またはVisual Studio をお使いください。
