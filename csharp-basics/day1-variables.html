C# の変数とデータ型

C# プログラムの基本構造

// 名前空間とクラス（.NET 6以降はトップレベルステートメントも可） using System; class Program { static void Main(string[] args) { Console.WriteLine("Hello, C#!"); } }

変数の宣言と型

C#は静的型付け言語。変数宣言時に型を指定する必要があります。

// 整数 int age = 25; long population = 1000000000L; // 小数 double price = 1980.5; decimal tax = 0.1m; // 金融計算には decimal を使う // 文字列 string name = "田中太郎"; char initial = 'T'; // 真偽値 bool isActive = true; bool isDeleted = false;

var キーワード（型推論）

var count = 10; // int と推論される var message = "Hello"; // string と推論される var pi = 3.14; // double と推論される
ポイント: var は型推論であり、動的型ではありません。 宣言時に型が確定し、後から変更はできません。

文字列操作

string firstName = "太郎"; string lastName = "山田"; // 文字列補間（推奨） string fullName = $"{lastName} {firstName}"; Console.WriteLine(fullName); // 山田 太郎 // 文字列の長さ Console.WriteLine(name.Length); // 4 // 大文字・小文字 string upper = "hello".ToUpper(); // HELLO string lower = "WORLD".ToLower(); // world

型変換

// 文字列 → 数値 string numStr = "42"; int num = int.Parse(numStr); int num2 = Convert.ToInt32(numStr); // 数値 → 文字列 string str = 123.ToString(); // 安全な変換（失敗してもクラッシュしない） bool success = int.TryParse("abc", out int result); // success = false, result = 0
注意: int.Parse() は変換できない値を渡すと FormatException をスローします。 安全な変換には TryParse を使いましょう。

Console.WriteLine / Write

Console.WriteLine("改行あり"); Console.Write("改行なし"); Console.WriteLine($"名前: {name}, 年齢: {age}");

練習問題

問1 かんたん

以下のコードを読んで、出力結果を答えてください。
int x = 10; int y = 3; var result = x + y; Console.WriteLine(result); Console.WriteLine(result.GetType());
答えを見る
13 System.Int32 // var で宣言しても、int + int の結果は int なので // GetType() は System.Int32 を返します

問2 かんたん

空欄を埋めて、出力が「私はAlice、20歳です。」になるようにしてください。
string name = "Alice"; int age = ____; Console.WriteLine($"私は{____}、{age}歳です。"); // 出力: 私はAlice、20歳です。
答えを見る
string name = "Alice"; int age = 20; // ← 1つ目の空欄 Console.WriteLine($"私は{name}、{age}歳です。"); // ↑ 2つ目の空欄は name

問3 ふつう

以下のコードにはバグがあります。正しく修正してください。
string input = "3.14"; int num = int.Parse(input); // ← ここがおかしい Console.WriteLine(num);
答えを見る
string input = "3.14"; // "3.14" は小数なので double または decimal で受け取る double num = double.Parse(input); // または double num2 = Convert.ToDouble(input); Console.WriteLine(num); // 3.14

問4 ふつう

以下の変数宣言を var を使って書き直してください。
int count = 100; string message = "完了"; bool isDone = true;
答えを見る
var count = 100; // int と推論される var message = "完了"; // string と推論される var isDone = true; // bool と推論される // var を使っても型は宣言時に確定する // count = "文字列"; とすると型が違うのでコンパイルエラーになる

問5 むずかしい

自分で書いてみよう！
名前・年齢・出身地を変数で宣言し、文字列補間を使って 「私は○○出身の△△歳、□□です。」と出力するコードを書いてください。
答えを見る
using System; class Program { static void Main(string[] args) { // 変数を宣言する string name = "田中太郎"; int age = 25; string city = "東京"; // 文字列補間で出力する Console.WriteLine($"私は{city}出身の{age}歳、{name}です。"); // 出力: 私は東京出身の25歳、田中太郎です。 } }
